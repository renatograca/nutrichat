# NutriChat Backend

Bem-vindo à documentação do backend do projeto NutriChat — uma API em FastAPI que implementa RAG (Retrieval-Augmented Generation) usando PostgreSQL + pgvector para armazenamento de documentos e embeddings.

## Visão rápida

- Framework: FastAPI
- Servidor de desenvolvimento: Uvicorn
- Banco de dados: PostgreSQL (+ pgvector)
- Propósito: ingestão de PDFs, indexação por embeddings e respostas via RAG
